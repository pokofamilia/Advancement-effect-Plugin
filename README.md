# Custom Advancement Plugin

Minecraftサーバーで進捗を達成した際、プレイヤー全員にスタイリッシュな通知を行うSpigot/Paper向けプラグインです。

##  特徴 (Features)

* **画面通知**
  進捗達成時に、画面中央（タイトル＆サブタイトル）に達成プレイヤーと進捗名を表示します。
* **進捗状況の可視化 (アクションバー)**
  画面下部に「進捗状況: X / 118」のように、現在の達成状況をリアルタイムで表示します。
* **Tabリストへの連携 (Scoreboard)**
  Tabキー（プレイヤーリスト）を押すと、各プレイヤーの名前の横に進捗の累計達成数が表示されます。ログイン時にも自動で同期されます。
* **データパックとの完全な互換性**
  進捗共有データパック等を導入し、複数人が同時に達成した際の「通知スパム」を防ぐ処理（0.5秒のクールダウン）を搭載しています。
* **フィルタリング**
  レシピ解放など、裏側で処理される非表示のバニラ進捗は自動でスキップされるため、ゲームプレイを邪魔しません。

##  動作環境 (Requirements)

* **Minecraft Version**: 1.20 / 1.21+ (Adventure APIをサポートしているサーバーを推奨)
* **Server Software**: Paper, Purpur, Spigot など

##  導入方法 (Installation)

1. `target` フォルダ内の `CustomAdvancementPlugin-1.0.jar` （またはご自身でビルドしたjar）をダウンロードします。
2. サーバーの `plugins` フォルダに配置します。
3. サーバーを起動、または再起動（`stop` -> `start`）します。
4. 設定は不要です。導入するだけで自動的にすべての機能が有効になります！
